# Android-Food_Delivery-App
* ordering with cart
* keeping order history
* download list of goods with details
* sign up, sign in

## Used technologies
* attaching firebase list of items to recycler view
* SQlite
* Firebase auth
* collapsing toolbar
* navigation bar
* card views
